## 什么是注解

注解是以"@注解名"在代码中存在的，并且可以添加一些参数值，例如@SuppressWarnings(value="unchecked"),注解可以附加在package,class,method,field等上面，相当于给他们添加了额外的辅助信息，通过这些辅助信息可以通过反射机制编程实现对这些元数据的访问。
